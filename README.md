# JavaScript Counter App

This is a simple counter application built using HTML, CSS, and JavaScript. The project demonstrates basic JavaScript concepts such as DOM manipulation, event handling, and UI interaction.

## Features

- Increase counter value
- Decrease counter value
- Reset counter value
- Clean and modern user interface
- Smooth button animations
- Beginner-friendly project

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JavaScript)

## How It Works

- The counter value increases when the Increase button is clicked.
- The counter value decreases when the Decrease button is clicked.
- The counter resets to zero when the Reset button is clicked.
